Pycharm Mac常用快捷键

opt + comm + L 格式化代码

comm + shift + V 从最近缓冲区粘贴

tab 缩进代码 tab + shift 反缩进

comm + P 显示方法的参数

comm + / 注释

comm + D 复制当前行

comm + Y 删除当前行

opt + shift + up/down 上移/下移当前行

comm + shift +up/down 上移/下移当前代码块

ctr + shift + J 智能拼接成一行

shift + enter 开始新的一行

comm + shift + U 切换大小写

opt + del +fn 删除到单词的末尾

opt + del 删除到单词的开头

comm + -/+ 折叠/展开代码块

comm + shift + F 全局查找

comm + F 文件内查找

双击shift 查询任何东西

comm + shift + -/+ 折叠/展开所有代码块

comm + R 文件内替换

comm + shift + R 全局替换

comm + K 提交代码到版本控制器

comm + T 从版本控制器更新代码

ctr + up + 左/右 切换当前桌面
